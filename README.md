# Startup Mentor

An Android application designed to connect entrepreneurs with mentors, manage sessions, and track startup growth.

## Project Details
- **Package Name:** `com.example.startupmentor`
- **Tech Stack:** Kotlin, MVVM, View Binding, Navigation Component
- **Minimum SDK:** 24
- **Target SDK:** 35

## Key Features
- **Onboarding Flow:** Interactive guide for new users.
- **Authentication:** Login and Registration system.
- **Mentor Discovery:** Browse and view detailed mentor profiles.
- **Session Management:** Book and track mentoring sessions.
- **Chat System:** Communication between mentors and mentees.
- **User Profiles:** Manage personal and professional startup info.

## Project Structure
- `app/src/main/java/com/example/startupmentor/` - Contains all Activity and Logic files.
- `app/src/main/res/layout/` - Contains all UI XML layouts.
- `gradle/libs.versions.toml` - Centralized dependency management.

## Setup & Run
1. Clone the repository.
2. Open in Android Studio (Ladybug or newer).
3. Sync Gradle and run on an emulator or physical device (API 24+).
